# @your-org/forms-themes

Phase 0 placeholder package for optional starter theme assets.

## Responsibility

This package will eventually provide removable starter tokens, CSS variables, class hooks, and default theme assets aligned with `DESIGN.md`.

## Boundary

Themes are optional. Core behavior must not depend on this package, and host apps must be able to replace or avoid default styling.

## Phase 0 Status

No theme behavior is implemented here yet. The current source entrypoint is only a bootstrap placeholder so the monorepo can typecheck, test, and build.
